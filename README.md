📦 电商 SKU 多功能计算器
这是一个专为电商卖家、采购人员和物流专员设计的轻量级 Web 计算工具。它集成了 SKU 记忆、订单总价计算、包装体积重计算以及常用单位换算四大功能。

界面经过全新美化，采用现代化的圆角卡片式设计（Card UI），视觉柔和，操作直观，且完全离线运行。

✨ 主要功能
🏷️ SKU 智能记忆：

支持录入 SKU 名称、单价和重量并保存到浏览器。

在订单表中输入 SKU 名称时，自动填充单价和重量，极大提高制表效率。

🧾 订单汇总计算：

动态添加/删除行。

自动计算单行总价。

底部实时汇总总数量、总重量（kg）和总金额。

📦 包装与体积重计算：

输入单个产品尺寸，通过设置长/宽/高叠加数量，自动计算最终装箱尺寸。

自动计算体积重（默认系数 /5000）和实重。

红字高亮关键输入项，结果一目了然。

🛠️ 实用转换工具：

尺寸：厘米 (cm) ⇋ 英寸 (inch) / 英尺 (ft) ⇋ 米 (m)。

重量：磅 (lb) ⇋ 克 (g) / 盎司 (oz) ⇋ 克 (g)。

液体：液盎司 (fl oz) ⇋ 毫升 (ml)。

性价比：快速计算“总价/数量”得出单价。

🚀 快速开始
本项目为单文件应用，无需安装任何服务器或环境。

下载或保存代码为 calculator.html。

双击文件，使用任意现代浏览器（Chrome, Edge, Safari, Firefox）打开。

即可立即开始使用，无需联网。

📖 使用指南
1. SKU 记忆功能
在页面顶部的 “SKU 记忆” 卡片中，输入产品名称（如 iPhone 15 Case）、单价和单个重量。

点击 “保存SKU” 按钮。

下次在下方表格的“名称/SKU”一栏输入该名称（或点击下拉选择），系统会自动填入价格和重量。

2. 订单汇总表
点击 “+ 添加一行” 增加条目。

输入或选择 SKU 后，补充“数量”，总价会自动计算。

底部黄色区域会显示整张单据的总金额和总预估重量。

3. 包装尺寸计算
这是为了模拟发货装箱场景：

第一行（单个尺寸）：输入产品本身的包装长宽带。

第二行（红色数字）：输入你想在长、宽、高三个方向分别叠加多少个产品（例如：长叠2个，宽叠3个，高叠4个）。

结果：系统会自动算出最终大箱子的尺寸、总个数，并对比体积重和实重。

4. 离线与隐私
所有数据（保存的 SKU）均存储在您浏览器的 LocalStorage 中。

数据不会上传到任何服务器，隐私绝对安全。

注意：如果您清理了浏览器缓存，保存的 SKU 记录可能会丢失。
